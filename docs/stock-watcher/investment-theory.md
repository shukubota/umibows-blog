## 1. 共通テクニカル分析

どちらの市場でも有効な手法。

### 1.1 移動平均線 (Moving Average)

**単純移動平均 (SMA)**

```
SMA_n = (P_1 + P_2 + ... + P_n) / n
```

- **短期 (5日・25日)**: ノイズを含むが反応が早い
- **中期 (75日)**: トレンドの方向性判断に使われる
- **長期 (200日)**: 強気/弱気相場の境界線として機能

**指数移動平均 (EMA)**

```
EMA_t = P_t × k + EMA_{t-1} × (1 - k)     k = 2 / (n + 1)
```

直近の価格に大きな重みを置くため SMA より早く反応する。
MACD などの派生指標の基礎として使われる。

**ゴールデンクロス / デッドクロス**

| クロス | 条件 | 意味 |
|--------|------|------|
| ゴールデンクロス | 短期 MA が長期 MA を**上抜け** | 上昇トレンド転換シグナル |
| デッドクロス | 短期 MA が長期 MA を**下抜け** | 下降トレンド転換シグナル |

注意: クロス後に逆行する「ダマシ」が発生しやすいため、出来高や他指標と組み合わせて判断する。

---

### 1.2 RSI (Relative Strength Index)

```
RS  = 直近 n 日間の平均上昇幅 / 直近 n 日間の平均下落幅
RSI = 100 - 100 / (1 + RS)
```

- 標準パラメータ: n = 14
- **RSI ≤ 30**: 過売り（売られすぎ）→ 反発期待
- **RSI ≥ 70**: 過買い（買われすぎ）→ 調整リスク
- **ダイバージェンス**: 価格が高値更新しているのに RSI が切り下がる場合 → 天井圏のシグナル

---

### 1.3 MACD (Moving Average Convergence Divergence)

```
MACD Line    = EMA_12 - EMA_26
Signal Line  = EMA_9 of MACD Line
Histogram    = MACD Line - Signal Line
```

- **MACD Line が Signal Line を上抜け**: 買いシグナル
- **MACD Line が Signal Line を下抜け**: 売りシグナル
- **ゼロライン上抜け**: 強気トレンド確認
- ヒストグラムが縮小傾向の場合はトレンド減速を示す

---

### 1.4 ボリンジャーバンド (Bollinger Bands)

```
中心線 = SMA_20
上限   = SMA_20 + 2σ
下限   = SMA_20 - 2σ
```

- 価格がバンド内に収まる確率は約 95%
- **バンドウォーク**: 上限/下限に沿って推移 → 強いトレンドを示す
- **スクイーズ**: バンドが収縮 → 大きな値動きの予兆（方向は不定）
- **エクスパンション**: バンドが拡大 → トレンド加速

---

### 1.5 出来高分析

- 価格上昇 + 出来高増加: トレンド継続の信頼性が高い
- 価格上昇 + 出来高減少: 上昇の勢いが失われつつある（天井警戒）
- 価格急落 + 出来高急増: 底打ちの可能性（セリングクライマックス）

---

### 1.6 ローソク足パターン

| パターン | 形状 | シグナル |
|---------|------|---------|
| 陽の丸坊主 | ひげなし・長い実体（陽線） | 強い買い圧力 |
| 陰の丸坊主 | ひげなし・長い実体（陰線） | 強い売り圧力 |
| 十字線 (ドージ) | 実体がほぼゼロ | トレンド転換の迷い |
| 包み線 (エンガルフィング) | 前の足を包む大きな実体 | 反転シグナル |
| 星 (スター) | ギャップを空けた小さい足 | 天井・底の転換点 |
| ハンマー / 逆ハンマー | 下ひげが実体の 2 倍以上 | 底打ち反発期待 |

## 2. 日本株（日経）

### 2.1 市場特性

| 項目 | 詳細 |
|------|------|
| 取引時間 | 前場: 9:00〜11:30 / 後場: 12:30〜15:30 |
| 決算サイクル | 3 月期決算が中心（4 月〜翌 3 月） |
| 主要指数 | 日経平均株価 (Nikkei 225) / TOPIX |
| 通貨影響 | 円安 → 輸出株に有利（自動車・電機） |
| 外国人比率 | 東証売買代金の約 70% を外国人が占める → 円/ドル・米国市場の影響を強く受ける |

---

### 2.2 日本株固有のシグナル

#### 決算サイクルと株価

```
本決算 (3月末)  → 5月上旬に開示
中間決算 (9月末) → 11月上旬に開示
第1四半期     → 8月上旬
第3四半期     → 2月上旬
```

- **決算プレイ**: 決算直前に仕込み、サプライズ期待で上昇、開示後に利確
- **決算ギャップ**: 良い決算でも「期待先取り」で下落するケースがある（Sell the fact）

#### 配当・優待の権利確定日

日本株特有の強い催促要因。

- **権利確定日**: 多くは 3 月末・9 月末の月末最終営業日
- **権利付き最終日**: 確定日の 2 営業日前（この日までに保有する必要がある）
- **権利落ち日**: 翌営業日に理論上は配当相当額だけ株価が下落

株価パターン:
```
権利付き最終日に向けて上昇 → 権利落ち日に急落 → 徐々に回復
```

#### 月次売上・生産指標

- 自動車: 毎月 1 日に前月販売台数発表 → トヨタ・ホンダ等に即日影響
- 百貨店: 毎月 10 日前後に前月売上発表 → 内需消費動向の先行指標
- 機械受注: 毎月中旬に経産省発表 → 設備投資・景気の先行指標（2 ヶ月後の設備投資を反映）

#### 日銀政策・為替連動

```
円安傾向 → 輸出株 (7203.T トヨタ, 6758.T ソニー) に追い風
円高傾向 → 内需株 (9983.T ファストリ, 7974.T 任天堂) に相対優位
```

日銀政策決定会合（年 8 回）はサプライズ発言が為替を急変させるため要注意。

---

### 2.3 日本株の季節性アノマリー

| 時期 | 傾向 | 背景 |
|------|------|------|
| 1 月効果 | 上昇傾向 | 新年の買い、機関投資家の新規設定 |
| 3〜4 月 | 不安定 | 決算確定売り・新年度買いが交錯 |
| 5 月 | 下落傾向 | "Sell in May" — 外国人の持ち高整理 |
| 8 月 | ボラティリティ高 | 閑散相場・決算集中・夏の外部ショック |
| 10〜11 月 | 上昇傾向 | 機関投資家の年末に向けた組み入れ |
| 12 月 | 節税売り後に反発 | 損出し確定 → 年末ラリー |

---

### 2.4 日本株特有のスクリーニング指標

**PBR (株価純資産倍率)**

```
PBR = 株価 / 1株あたり純資産 (BPS)
```

東証が 2023 年に PBR 1 倍割れ企業へ改善要求を出したことで注目度が高まった。
- PBR < 1.0: 理論上は解散価値以下（バリュー株候補）
- 自社株買い・配当増配で PBR 改善を迫られている企業が多い

**配当利回り**

```
配当利回り = 1株あたり配当 / 現在株価
```

- 3% 以上: インカム投資として魅力的
- 長期金利との比較で相対評価する（金利が上がると配当株の魅力が下がる）

## 3. 米国株

### 3.1 市場特性

| 項目 | 詳細 |
|------|------|
| 主要取引所 | NYSE / NASDAQ |
| 取引時間 | 東部時間 9:30〜16:00 (日本時間: 夏 22:30〜翌 5:00 / 冬 23:30〜翌 6:00) |
| プレ・アフター | 取引前後 4 時間の時間外取引あり（ボラティリティ高） |
| 決算サイクル | 四半期ごと（1/4/7/10 月が決算シーズン） |
| 主要指数 | S&P 500 / NASDAQ 100 / Dow Jones 30 |

---

### 3.2 米国株固有のシグナル

#### 決算シーズン (Earnings Season)

```
Q1 決算: 4 月中旬〜5 月上旬
Q2 決算: 7 月中旬〜8 月上旬
Q3 決算: 10 月中旬〜11 月上旬
Q4 決算: 1 月中旬〜2 月上旬
```

重要指標:
- **EPS**: 1 株あたり利益。アナリスト予想との乖離が株価を動かす
- **Revenue**: 売上高。EPS より先行する成長指標
- **ガイダンス**: 次四半期・通期見通し。ポジティブサプライズでも将来見通しが弱ければ急落する

#### FRB・FOMC の影響

年 8 回の FOMC 会合が全市場に影響する。

```
利上げ → グロース株 (ハイテク) に逆風、バリュー株・金融株に追い風
利下げ → グロース株・不動産・長期債に追い風
```

- **ドットプロット**: 年 4 回公表される委員の金利予測 → 市場の中期見通しに影響
- **パウエル議長会見**: 発言のトーンで数百億ドル規模の資金移動が起きる

#### 経済指標カレンダー

| 指標 | 頻度 | 影響度 |
|------|------|--------|
| CPI (消費者物価指数) | 毎月中旬 | 最高 — FRB の利上げ判断に直結 |
| 雇用統計 (NFP) | 毎月第 1 金曜 | 最高 — 景気・利上げの両面に影響 |
| GDP | 四半期ごと | 高 — 景気サイクルの確認 |
| PMI (購買担当者指数) | 毎月初め | 中〜高 — 製造業・サービス業の先行指標 |
| PPI (生産者物価指数) | 毎月中旬 | 中 — CPI の先行指標 |
| 小売売上高 | 毎月中旬 | 中 — 消費動向の確認 |

**判断の基準:**
```
CPI 予想比 +0.1% → 金利上昇圧力 → ハイテク株売り
雇用統計 非農業部門 予想比 +10万人 → 好景気だがインフレ懸念 → 混在シグナル
```

---

### 3.3 セクターローテーション

景気サイクルに応じてリード・ラグするセクターが変わる。

```
景気回復期   → 素材・工業・エネルギー
景気拡張期   → 一般消費財・情報技術・金融
景気後退警戒 → ヘルスケア・生活必需品・公益
景気後退期   → 債券・公益・金（ゴールド）
```

**セクター ETF での監視**:
- XLK (情報技術) / XLE (エネルギー) / XLF (金融) の相対強度を比較
- リード・ラグ関係が逆転した場合は転換期のシグナル

---

### 3.4 VIX (恐怖指数)

```
VIX = S&P 500 オプションが織り込む今後 30 日間のボラティリティ（年率換算）
```

| VIX 水準 | 市場の状態 |
|---------|-----------|
| < 15 | 楽観・低ボラティリティ（過熱警戒） |
| 15〜20 | 通常レンジ |
| 20〜30 | 警戒感の高まり |
| > 30 | 恐慌的売り（逆張り買いの機会になりやすい） |

- VIX スパイクはしばしばセリングクライマックスと同時に起き、短期底打ちの逆張り指標になる
- VIX が低位で安定している場合は過信に注意（低ボラティリティは次のショックへの脆弱性を高める）

---

### 3.5 オプション市場のシグナル

**Put/Call レシオ**

```
Put/Call Ratio = Put オプション出来高 / Call オプション出来高
```

- > 1.0: Put（下落ヘッジ・弱気）が多い → 過度な悲観 → 逆張り買いシグナル
- < 0.7: Call（上昇期待・強気）が多い → 過度な楽観 → 過熱警戒

**オプションの満期 (OpEx)**

毎月第 3 金曜日はオプション満期日。大量ポジションの巻き戻しが短期的な値動きをゆがめる。
四半期末（3/6/9/12 月）の OpEx は特にボラティリティが高い。

**ガンマスクイーズ**

大量の OTM Call が購入されると、マーケットメーカーがデルタヘッジで株を買い続け、価格上昇が加速するフィードバックループが起きる（2021 年 GME などが代表例）。

---

### 3.6 米国株の季節性アノマリー

| 時期 | 傾向 | 背景 |
|------|------|------|
| 1 月効果 | 小型株上昇 | 年末の税務上の損出し売りが 1 月に買い戻される |
| 5〜9 月 | 割安 | "Sell in May and go away" — 過去統計的に低パフォーマンス |
| サンタクロースラリー | 12 月下旬〜1 月上旬上昇 | 薄商い・機関投資家の休暇中に小型株が上昇しやすい |
| 大統領選挙サイクル | 3 年目に最も強い | 再選を意識した財政出動が多い |

---

### 3.7 米国マクロ指標と株価の関係

**長短金利差 (イールドカーブ)**

```
スプレッド = 10年債利回り - 2年債利回り
```

- スプレッド < 0 (逆イールド): 過去 60 年で 100% の確率でその後 景気後退が来ている（ただしラグは 6〜24 ヶ月）
- スプレッドが再びプラスに戻るタイミングが実際の後退が始まるサイン

**ドル指数 (DXY)**

```
DXY 上昇 → 新興国・コモディティ・多国籍企業に逆風
DXY 下落 → 新興国株・コモディティ・輸出依存企業に追い風
```

**M2 マネーサプライ**

- M2 増加 → 市場に資金が溢れ株価が上昇しやすい
- FRB QT (量的引き締め) による M2 縮小は株式市場の流動性を奪う

## 4. シグナル優先度と組み合わせ

### 4.1 信頼度マトリクス

| シグナル | 単独信頼度 | 組み合わせ効果 |
|---------|-----------|--------------|
| ゴールデンクロス (MA5/25) | 中 | 出来高増加と組み合わせで高 |
| RSI 過売り | 中 | トレンドが下降中の場合は無効になりやすい |
| MACD クロス | 中 | ゴールデンクロスと同時発生で高 |
| ボリンジャー下限タッチ | 中 | RSI 過売りと組み合わせで高 |
| 価格アラート | 高 | 事前に設定した重要価格での反応を見る |
| 決算ビート + ガイダンス引き上げ | 高 | 翌日のギャップアップ継続を確認する |

### 4.2 エントリー基準の例（複数シグナル重複）

**強気エントリー条件（日本株）:**
```
条件1: RSI ≤ 35
条件2: 価格がボリンジャー下限以下
条件3: 出来高が 20 日平均の 1.5 倍以上
→ 3 条件すべて満たした場合のみ「強い買いシグナル」として通知
```

**慎重モード（米国株）:**
```
- VIX > 25 の場合はシグナルを通知のみ（推奨しない旨を付記）
- 逆イールド継続中はグロース株シグナルの信頼度を下げる
- FRB 会合・雇用統計の 24 時間前後はシグナル判断を保留
```

### 4.3 損切り・利確ライン

シグナル通知時は参考の損切り・利確ラインも合わせて表示する。

```
買いシグナル時:
  損切り基準: エントリー価格 - 直近安値 の差分 × 1.5
  利確目標:  エントリー価格 + (損切り幅 × 2〜3)   ← リスクリワード 1:2〜3
```
