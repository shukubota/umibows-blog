# プラスにするための投資原則

テクニカル指標を「知っている」だけでは勝てない。
このドキュメントは「何をすれば勝率が上がるか」だけに絞る。
根拠のない手法・ノイズになる情報は意図的に除外している。

---

## 絶対ルール（これを守れないなら始めない）

### 1. 損切りラインを先に決めてから買う

```
エントリー前に決める:
  損切り価格 = エントリー価格 × (1 - 損切り率)
  損切り率: 個別株 5〜8% / ETF 3〜5%
```

損切りは「負けを認める」ではなく「資金を守る」行為。
含み損を抱えたまま「戻るのを待つ」は最悪のパターン。戻らないことがほとんど。

### 2. 1トレードのリスクを資金の 1〜2% 以内に抑える

```
例: 資金 100万円、損切り率 5% の場合
  最大損失許容: 100万 × 2% = 2万円
  購入可能額   = 2万円 ÷ 5% = 40万円分まで
```

資金の 1〜2% ルールを守れば、10連敗しても資金の 80% 以上が残る。
これがないと1回の失敗で再起不能になる。

### 3. リスクリワード比 1:2 以上の取引しかしない

```
損切り幅: 5%
利確目標: 10% 以上（リスクリワード 1:2）
```

勝率 40% でもリスクリワード 1:2 なら長期プラスになる:

```
10回中 4勝6敗、利確 +10%、損切り -5%
期待値 = 4 × 10% - 6 × 5% = +10%
```

## 実証されているシグナル（ノイズが少ない順）

### A. 決算サプライズ後のドリフト（PEAD）★★★

**最も信頼度が高い。学術論文で何十年も確認されている anomaly。**

```
条件:
  ① EPS が市場予想を 10% 以上上回る（正の決算サプライズ）
  ② ガイダンス（次期見通し）も引き上げ
  ③ 決算翌日のギャップアップが出来高増加を伴う

タイミング:
  決算翌日の寄り付きにエントリー
  その後 1〜4 週間のドリフト上昇を狙う
```

なぜ機能するか: 市場は決算サプライズを即座に完全に織り込まない。
機関投資家が徐々に持ち高を積み増す過程で価格が上昇し続ける。

**除外条件（ダマシを避ける）:**

- 決算前に株価が 30% 以上上昇済みの場合（期待先取り済み）
- VIX > 25 の地合いが悪い時期
- ギャップアップしたが出来高が平均以下の場合

---

### B. トレンドフォロー（移動平均フィルター付き）★★

単なるゴールデンクロスではなく、**市場環境フィルターをかけてから使う**。

```
エントリー条件（すべて満たす場合のみ）:
  ① 日足 200日 MA より株価が上にある（上昇トレンド確認）
  ② MA5 が MA25 を上抜け（ゴールデンクロス）
  ③ 出来高が 20日平均の 1.5倍以上（本物のブレイクアウト確認）

除外:
  株価が 200日 MA より下にある場合は一切エントリーしない
  （下落トレンドの中でのゴールデンクロスはほぼダマシ）
```

200日 MA フィルターだけでダマシのゴールデンクロスの 60% 以上を除外できる。

---

### C. モメンタム（3〜12ヶ月の相対強度）★★

```
計算:
  過去 6ヶ月のリターンで銘柄をランキング
  上位 20% の銘柄を保持、下位 20% を空売り（または除外）
  毎月リバランス
```

なぜ機能するか: 上昇している銘柄は機関投資家の継続的な買いが入っている。
逆張りより順張りの方が統計的に有利（Jegadeesh & Titman 1993 で実証）。

**重要な注意:**

- 直近 1ヶ月は除外する（短期リバーサルが起きやすい）
- 市場全体が下落トレンドの時はモメンタム戦略をオフにする

---

### D. 過売り圏での逆張り（条件付き）★

RSI 単独では弱い。以下の条件が重なった場合のみ有効。

```
条件（すべて揃った時のみ）:
  ① RSI ≤ 30
  ② 株価がボリンジャー下限を下回っている
  ③ 長期上昇トレンド中（200日 MA の上）である
  ④ 急落の原因が業績悪化でなく市場全体の下落である

除外（これがあれば見送る）:
  - 決算ミス・不正・業績下方修正による急落
  - セクター全体の構造的問題（例: 規制変更）
  - 出来高が急増しているが理由が不明
```

下落トレンドの銘柄に逆張りするのは「落ちてくるナイフをつかむ」。
上昇トレンド中の一時的な押し目での逆張りにのみ使う。

## 使ってはいけない手法（ノイズ）

| 手法                       | 除外理由                                               |
| -------------------------- | ------------------------------------------------------ |
| ローソク足パターン単独     | 統計的に有意な優位性がほぼない                         |
| MACD 単独                  | ゴールデンクロスと同じ情報。重複                       |
| 短期 RSI 売買（デイトレ）  | 取引コストで期待値がマイナスになる                     |
| ニュース・SNS ベースの売買 | 情報が届いた時点で価格に織り込まれている               |
| PER・PBR 単独の割安投資    | 「割安には理由がある」ことが多い。触媒がないと動かない |
| 決算前の仕込み             | 不確実性が高すぎる。方向が読めない                     |
| 損切りしない塩漬け         | 資本効率を著しく下げる                                 |

## 市場環境の判断（地合いフィルター）

シグナルの有効性は市場環境で大きく変わる。先に地合いを判断してからシグナルを使う。

```
地合い判断フロー:

① S&P500（米）または TOPIX（日）が 200日 MA の上？
  YES → 通常モード（A〜D のシグナルを使う）
  NO  → 守りモード（新規エントリーをしない・現金比率を上げる）

② VIX > 30？
  YES → 逆張り注意（急落局面。底を当てようとしない）
  NO  → 通常モード継続

③ 逆イールド（10年債 - 2年債 < 0）が 3ヶ月以上継続中？
  YES → グロース株の比率を下げ、ディフェンシブ・キャッシュを増やす
  NO  → 通常モード継続
```

## バックテストの必須手順

シグナルを実装する前に必ずバックテストで検証する。

```python
# 最低限確認する指標
勝率          (Win Rate)    # 目安: 40% 以上あれば戦略次第でプラス可
平均リスクリワード          # 目安: 1:2 以上
最大ドローダウン            # 目安: 20% 以内（これ以上は精神的に継続困難）
シャープレシオ              # 目安: 1.0 以上（リスク調整後リターン）
トレード数                  # 最低 100 トレード以上（サンプル不足は信頼できない）
```

バックテストがプラスでも以下の点を確認する:

- **過学習チェック**: パラメータを少し変えても結果が大きく変わる場合は過学習
- **アウトオブサンプル**: 直近 20% のデータはバックテストに使わず、戦略完成後に検証用として使う
- **取引コスト込み**: 税・手数料を必ず差し引いて計算する

## 実践フロー（これだけ守れば戦える）

```
毎週末にやること:
  1. TOPIX / S&P500 が 200日 MA の上か確認（地合いチェック）
  2. 保有銘柄の損切りラインを再確認
  3. 決算スケジュールを確認（翌週の決算発表銘柄リスト）

エントリー前チェックリスト:
  □ 200日 MA の上にある
  □ 損切り価格を決めた
  □ リスクリワードが 1:2 以上
  □ 1トレードのリスクが資金の 2% 以内
  □ 地合いフィルターを通過した

やらないこと:
  × 感情で損切りを後ろ倒しにする
  × 「もう少し待てば戻る」という思考
  × 1銘柄に資金の 20% 以上を集中させる
  × バックテストなしで新しいシグナルを本番適用する
```

## 期待値の現実的な見積もり

上記を全て守った場合の現実的な目標:

| 条件                              | 年率リターン目安                       |
| --------------------------------- | -------------------------------------- |
| インデックス買いっぱなし (S&P500) | 7〜10%                                 |
| 上記ルールを守ったアクティブ運用  | 10〜20%（ただし時間コストあり）        |
| ルールを守らないアクティブ運用    | 市場平均以下（大半の個人投資家の現実） |

市場平均に継続的に勝ち続けることはプロでも難しい。
**目標は「大きく負けない」こと**。大きく負けなければ、小さい勝ちの積み上げが資産を増やす。
