# 米国株分析ダッシュボード

## 概要

米国株市場の現状を一覧できるWebダッシュボード。
売買機能はなし。「今市場がどういう状態か」を把握するための分析・可視化ツール。

---

## 画面構成

```
┌────────────────────────────────────────────────────────┐
│  [1] 市場概況バー（常時表示）                            │
│  S&P500 +1.2%  NASDAQ +0.8%  DOW -0.3%  VIX 18.4     │
├──────────────────────┬─────────────────────────────────┤
│  [2] セクター         │  [3] ウォッチリスト             │
│  ヒートマップ         │  個別銘柄 + テクニカル指標      │
│  (11セクター)         │                                 │

│  [4] 地合いパネル                                       │

├────────────────────────────────────────────────────────┤
│  [5] 決算カレンダー（今週の主要決算）                    │
└────────────────────────────────────────────────────────┘
```

## 各パネルの仕様

### [1] 市場概況バー

| 表示項目 | データソース | 更新頻度 |
|---------|------------|---------|
| S&P500 (SPY) 前日比 | yahoo-finance2 | ページロード時 |
| NASDAQ (QQQ) 前日比 | yahoo-finance2 | ページロード時 |
| Dow Jones (DIA) 前日比 | yahoo-finance2 | ページロード時 |
| VIX 現在値 | yahoo-finance2 (`^VIX`) | ページロード時 |
| ドル円 | yahoo-finance2 (`USDJPY=X`) | ページロード時 |

VIX の背景色:
- < 15: 緑（低ボラ）
- 15〜25: 黄（注意）
- > 25: 赤（警戒）

---

### [2] セクターヒートマップ

11セクター ETF の前日比をヒートマップで表示。
どのセクターに資金が流入/流出しているか一目で把握する。

| ティッカー | セクター |
|-----------|---------|
| XLK | 情報技術 |
| XLF | 金融 |
| XLV | ヘルスケア |
| XLE | エネルギー |
| XLI | 資本財 |
| XLY | 一般消費財 |
| XLP | 生活必需品 |
| XLB | 素材 |
| XLU | 公益 |
| XLRE | 不動産 |
| XLC | コミュニケーション |

色: 前日比 +2% 以上=濃緑 / +1%=薄緑 / 0%=グレー / -1%=薄赤 / -2%以下=濃赤

---

### [3] ウォッチリスト

`config/watchlist.yaml` で管理する銘柄を一覧表示。

各銘柄の表示カラム:
```
ティッカー | 現在値 | 前日比 | 200日MA比 | MA5/25 | RSI | 地合い判定
```

**地合い判定カラム（how-to-win.md のルールを自動適用）:**

| 判定 | 条件 |
|------|------|
| 強気候補 | 200日MA上 + RSI 40〜60 + MA5がMA25上 |
| 過買い注意 | RSI ≥ 70 |
| 過売り注目 | 200日MA上 + RSI ≤ 35 |
| トレンド外 | 200日MA下（エントリー非推奨を表示） |

クリックで個別銘柄の詳細チャート（価格 + MA + RSI）を展開表示。

---

### [4] 地合いパネル

市場全体の状態を `how-to-win.md` の地合いフィルターに基づいて表示。

```
[S&P500 vs 200日MA]  ● 上 (強気モード) / ○ 下 (守りモード)
[VIX水準]            18.4 — 通常レンジ
[イールドカーブ]      10年-2年: +0.12% — 正常
[Put/Call比率]        0.82 — やや強気
```

総合判定バナー:
- 「強気モード: 通常運用」
- 「守りモード: 新規エントリー非推奨」
- 「警戒: VIX高騰中」

---

### [5] 決算カレンダー

今週・来週の主要銘柄の決算予定を表示。

```
月曜  AAPL  予想EPS: $1.43  発表: 市場後
水曜  NVDA  予想EPS: $0.64  発表: 市場後
木曜  MSFT  予想EPS: $2.82  発表: 市場後
```

決算済み銘柄は実績EPS・サプライズ率（予想比）も表示。

## 技術スタック

| 役割 | 採用技術 | 理由 |
|------|---------|------|
| フレームワーク | Next.js 14 App Router | 既存プロジェクトと統一 |
| 株価取得 | `yahoo-finance2` (npm) | TypeScript ネイティブ、yfinance 相当 |
| テクニカル指標 | `technicalindicators` (npm) | RSI・MA を JS で計算 |
| チャート | `recharts` | 既存プロジェクトで使いやすい React 向け |
| スタイル | Tailwind CSS | 既存プロジェクトと統一 |
| データフェッチ | Next.js Server Components | APIルート不要でシンプル |

## 実装フェーズ

### Phase 1 — データ取得 & 表示
- [ ] `yahoo-finance2` で市場概況・セクター ETF・ウォッチリスト取得
- [ ] 市場概況バー（[1]）を実装
- [ ] ウォッチリスト一覧（[3]）をシンプルなテーブルで表示

### Phase 2 — 分析機能
- [ ] テクニカル指標計算（RSI / MA）
- [ ] セクターヒートマップ（[2]）を実装
- [ ] 地合いパネル（[4]）と自動判定ロジック

### Phase 3 — チャート & 決算
- [ ] 個別銘柄の価格チャート + MA + RSI（recharts）
- [ ] 決算カレンダー（[5]）

### Phase 4 — 拡張（オプション）
- [ ] 自動リフレッシュ（5分ごと）
- [ ] PEAD 候補銘柄のハイライト（決算サプライズ検出）

---

## 注意事項

- **データ遅延**: yahoo-finance2 は最大 15 分遅延（無料 API のため）
- **米国市場時間**: 東部時間 9:30〜16:00（日本時間 冬 23:30〜翌 6:00）
- **レートリミット**: 短時間に大量リクエストをすると Yahoo Finance にブロックされる。Server Components でまとめてフェッチし、適切にキャッシュする
